# Images Directory

This directory is for storing images used across the Aarya Hotel website.

## Image URLs Updated

All image references in the HTML files have been updated to point to:
```
https://www.aaryahotel.com/images/[image-name].jpg
```

## Required Images

To complete the website, you need to:

1. **Download images from www.aaryahotel.com**
2. **Optimize them for web** (recommended: WebP format, 70-85% quality)
3. **Replace the URLs** in HTML files with actual image paths
