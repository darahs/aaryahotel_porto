## Image Optimization Tips

1. **Format**: Use WebP or optimized JPEG
2. **Sizes**:
   - Hero images: 1920x1080px or larger
   - Card images: 800x600px
   - Thumbnails: 400x300px
3. **Compression**: Optimize for web (70-85% quality)
4. **Alt Text**: Always include descriptive alt text
5. **Lazy Loading**: Consider implementing lazy loading for better performance

## How to Add Images

1. Download images from www.aaryahotel.com
2. Optimize images using tools like:
   - TinyPNG
   - ImageOptim
   - Squoosh
3. Place in `images/` folder (create if needed)
4. Update image URLs in HTML files
5. Or use direct URLs from the hotel's website/CDN
