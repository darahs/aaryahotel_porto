# Aarya Hotel Image URLs

## Image URL Format

All images from Aarya Hotel website use this format:
```
https://www.aaryahotel.com/images/package/galleryimages/[FILENAME].webp
```

## Example Image URL
```
https://www.aaryahotel.com/images/package/galleryimages/N5XLf-2.webp
```

## How to Find More Images

1. Visit https://www.aaryahotel.com
2. Open browser developer tools (F12)
3. Go to Network tab
4. Filter by "Images"
5. Navigate through the website pages
6. Look for image requests to `/images/package/galleryimages/`
7. Copy the full image URLs

## Image Format

- **Format**: WebP (`.webp`)
- **Location**: `/images/package/galleryimages/`
- **Benefits**: WebP provides better compression and quality than JPEG

## Browser Support

WebP is supported by:
- Chrome (since version 23)
- Firefox (since version 65)
- Edge (since version 18)
- Safari (since version 14)
- Opera (since version 12.1)

For older browser support, consider using a fallback or converting to JPEG.
